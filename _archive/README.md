# Archived Reference Code

This folder contains frozen UI, providers, and mock data from the earlier version of the Tidal prototype (Home, Global Chat, Pool, Swap).

It is intentionally excluded from the TypeScript build and Next.js compilation via [tsconfig.json](../tsconfig.json) and [next.config.ts](../next.config.ts). Nothing under `_archive/` is imported from live code.

Kept here only so patterns, component structures, and mock scenarios can be copied back out later if needed. Do not edit in place — treat as read-only reference material.
